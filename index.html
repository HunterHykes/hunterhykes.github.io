H.A.L.O. - Hard Hat Ambient Liability Observer

Senior Design Project
This product is a hard hat attachment which utilizes time-of-flight (ToF) sensors to detect hazards near the wearer's head. Dependent on the distance between the user and the hazard, some RGB LEDs placed near the brim of the hard hat will turn on and change hue from yellow to red as objects approach. In the event that the user suffers an impact to their head, an on-board accelerometer will detect the impact severity. If this impact is determined to pose a concussion threat to the user, the accelerometer data is logged to a .csv file on a micro-SD card.
